## Key Features

| Feature                        | Description                                                                 |
|--------------------------------|-----------------------------------------------------------------------------|
| Offline Models                 | Phi-4, Qwen 2.5 Instruct, DeepSeek R1, Mistral — run locally after download |
| ONNX Runtime Acceleration      | Auto-detects CPU/GPU/NPU — 5–10× faster on M1–M4                            |
| OpenAI-Compatible API          | Integrate into apps with standard SDK — no code changes                     |
| Model Catalog                  | 20+ edge-optimized models from Hugging Face                                 |
| Privacy-First                  | All data stays on-device — no cloud, no tracking                            |
| Cross-Platform SDK             | Same code for macOS, Windows, mobile — scale to Azure later                 |
| CLI & Service Management       | Easy install via Homebrew • Restart service in one command                  |
| Hardware Optimization          | Metal GPU on Apple Silicon • Low latency for real-time apps                 |
